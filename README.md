# Tiffin Ordering System – Python

This project is a simple console-based Python application for ordering tiffin items such as dosa, idly, puri, pongal, etc.  
It calculates the total bill based on the selected items and their quantities.

## Features
- Menu with item prices
- User can place multiple orders
- Calculates bill automatically
- Displays total amount at the end
- Uses dictionary for menu handling

## Technologies Used
- Python

## How to Run

1. Download or clone the repository
2. Run the program using:

   python tiffen.py

3. Enter the item name and quantity.
4. Type `exit` to finish ordering and see the total bill.

## Sample Menu

- Dosa – ₹30  
- Idly – ₹30  
- Puri – ₹25  
- Pongal – ₹20  
- Vada – ₹20  
- Bread Omelette – ₹30
